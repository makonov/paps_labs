# Система детализации структуры затрат IT-компании

IT-компания хочет заменить большую Excel-модель, которая используется для расчёта стоимости сотрудников, трудозатрат, косвенных расходов и рентабельности проектов. Текущий Excel-файл содержит множество листов и макросов, работает медленно и требует ручного обслуживания. Компания планирует перенести основную логику расчётов в корпоративную BI-платформу, сохранив возможность формирования отчётов в привычном формате.

**Пользователи:** генеральный директор, главный бухгалтер и сотрудник по работе с документами.

## Требования
- Загрузка данных о сотрудниках из корпоративной ERP-системы  
- Загрузка дополнительных данных из плоских файлов  
- Расчёт внутренних ставок сотрудников  
- Расчёт рекомендуемых внешних ставок  
- Учёт трудозатрат по проектам  
- Расчёт косвенных затрат  
- Ведение историчности данных  
- Расчёт прибыли и рентабельности проектов  
- Формирование отчётов для пользователей  
- Автоматическое обновление данных  

## Дополнительный контекст
- Существующая Excel-модель слишком медленная.  
- Не поддерживает удобное хранение исторических данных.  
- Некоторый функционал ранее уже пробовали перенести на BI-платформу, но алгоритмы были реализованы неправильно, из-за чего система не используется.
